# 前端联调冒烟清单（阶段6 功能17）

## 环境准备
1. 启动后端：`cd backend && mvn spring-boot:run "-Dspring-boot.run.profiles=local"`
2. 启动前端：`cd frontend && npm run dev`
3. 准备测试数据：
   - 执行初始化：`mysql -u root -p < backend/db/init.sql`
   - 执行阶段数据：`mysql -u root -p < backend/db/seed_phase1.sql`

## 账号相关
1. 注册新账号，期望注册成功并跳转登录页。
2. 使用新账号登录，期望本地存储存在 `token`。
3. 使用错误密码登录，期望显示后端返回的错误信息。

## 电影浏览
1. 进入电影列表页，翻页/搜索/分类/排序均可用。
2. 进入电影详情页，刷新页面后数据仍正常展示。

## 用户行为
1. 已登录状态下在详情页评分，刷新后评分保持。
2. 已登录状态下收藏/取消收藏，刷新后状态保持。
3. 进入个人中心，确认“我的评分/我的收藏”来自后端数据。

## 推荐
1. 登录后进入推荐页，确认列表有推荐理由与匹配度。
2. 已评分或已收藏电影不应出现在推荐结果中。
3. 无可推荐数据时，页面显示空态提示。

## 后台管理
1. 普通用户进入后台页，期望提示“无权限访问后台（仅管理员）”。
2. 管理员账号进入后台页，期望可加载电影列表。
3. 管理员新增电影后，列表可见新数据。
4. 管理员编辑电影后，列表和详情页数据同步更新。

## 回归检查
1. 再次检查登录、评分、收藏、推荐、后台功能均可用。
2. 检查主要错误提示是否统一为后端 `message`。
